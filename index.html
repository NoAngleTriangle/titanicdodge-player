Titanic Dodge




Thanks to @NickNackTheTitaniac (scratch.mit.edu) for the original project Iceberg Dodge (scratch.mit.edu).

Instructions:
--=== TITANIC DODGE! ===--

WASD or arrow keys to move. Avoid the titanics and see if you can change the fate of the RMS Iceberg.
The titanics can be harder to see at night, so steer clear!

This game is mobile friendly, just turn on mobile mode in settings :)
